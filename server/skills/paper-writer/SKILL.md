# Paper Writer Skill

## Structure

Always write papers with:

1. Abstract
2. Introduction
3. Background / Literature Review
4. Methodology
5. Results / Findings
6. Discussion
7. Conclusion
8. References

## How to write

```bash
cat > /workspace/paper.md << 'EOF'
# Paper Title

## Abstract
...

## 1. Introduction
...
EOF
```

## Tips

- Research first, write second.
- Cite sources with URLs.
- Be specific with data and numbers.
- Save output as /workspace/paper.md.
